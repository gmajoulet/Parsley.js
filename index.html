Parsley.js

Parsley.js allows you to automatically validate your forms frontend, without writting actually a single line of javascript, thanks to its ultra simple API.

It strives to have a javascript agnostic interface for UI/UX developers that might want to use it. Just add some data-validate="parsley" in your form tags, some data-type="email", data-type="digits" to validate your fields, and you're good to go!

Requirements / Compatibiliy

Parsley is delivred to you in different builds (in the Github dist/ directory), here they are:

  • Parsley.min.js: ~8K, requires jQuery or Zepto (with data module builded for Zepto

  • Parsley-standalone.min.js: ~25K, with minimum Zepto config builded inside. Requires nothing more. Only use that one if you don't already have jQuery or Zepto on your website, could make conflicts

  • Tested on only recent browsers at the moment
    Use it that way with Parsley: 
    <script src="http://code.jquery.com/jquery-1.8.2.min.js"></script>
<script src="parsley.js"></script>

Usage

  • Include jQuery or Zepto (if not already here) and parsley.js
    <script src="jquery.js">
<script src="parsley.js">
  • Add data-validate="parsley" to the forms you want to auto-validate
    <form data-validate="persist" method="POST">
    <input type="text" data-type="email" />
</form>

That's all! ;)
